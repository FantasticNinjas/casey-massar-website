Musical Floppy Drives

A pair of unassuming floppy drives, a few cables, and a big red button. When the button is pressed, the floppy drives jump into action, playing a rather fitting tune.

As strange and impressive as this project is (especially in person), the principal behind it is actually pretty simple. Data on a floppy disk is stored in rings (called tracks). A floppy drive spins the disk in order to move along a track and uses a stepper motor to move the head back and forth in order to switch between tracks. It is this stepper motor that produces the music. Each time the stepper motor steps it produces a soft click (a sound you will be familiar with if you've ever owned a computer with a floppy drive). Step the motor fast enough and those clicks blend together to form a tone.

This project uses one of the timer/counter units on an AVR microcontroller to step the motor in each floppy drive at a precicely controlled frequency. It adjusts the frequency of each drive according to a pre-programmed sequence, producing a song. My source code can be downloaded at the bottom of the page.

Materials used on this project:
  • Atmel ATmega328p Xplained Mini dev board
  • 2 3 ½ floppy drives
  • ATX power supply (A full ATX power supply is not really necessary for this, any 5V power supply will do. But I had this lying around and it already had the connectors for floppy drives.)
  • Jumper wires

Credit where credit's due, I am not the first person to build musical floppy drives. As far as I have been able to dissern, that title goes to YouTube user Sammy1Am. However, the code linked here is entirely my own.
